Michael Toomim and the Invisible College Anyone can add a website to the open web, but individual websites are closed. And the economics of the web have led to a number of large near-monopoly websites, such as Facebook and Google. To compete with these monopolies, we need open websites that anyone can contribute to. We can build open websites with a new web architecture that defines a standard protocol for the insides of websites. This architecture, called Statebus, both makes it easier to develop the internals of a website, while simultaneously making it easy to share the internals with other websites. Caption:
  1. Before the web, people could connect to monolith networks like Aol, Compuserve and Prodigy. Each had a proprietary protocol. To contribute to the monolith, you needed special permission. Monoliths did not link to one another.
  2. On the web, every page has a URL, and can link to any other page. Websites add value to one another by linking to each other. But they can only link to pages: the presentation of the underlying data in a database; the surface of the websites.
  3. Statebus opens the insides of websites. Each piece of state has a URL. A website's state can derive from other websites.
